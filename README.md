🧠 Medical Image Classifier
A professional Web-based diagnostic assistance tool built with Streamlit and TensorFlow. This application allows users to upload medical images (MRI scans or Chest X-rays) and receive instant classification results using pre-trained deep learning models.

🚀 Features
Dual Model Support: Toggle between Brain Tumor detection and Pneumonia screening.

Real-time Inference: Fast processing using optimized .keras model weights.

Interactive UI: Clean, modern interface with image previews and confidence scores.

Dynamic Background: Custom-styled CSS for a polished, medical-grade aesthetic.

🛠️ Tech Stack
Frontend: Streamlit

Deep Learning: TensorFlow/Keras

Image Processing: Pillow (PIL)

Data Handling: NumPy

📋 Prerequisites
Ensure you have Python 3.9+ installed and the following dependencies:

streamlit

tensorflow

numpy

Pillow
